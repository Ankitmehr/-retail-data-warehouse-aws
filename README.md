# 🏪 Retail Data Warehouse — AWS + Power BI

## 📌 Project Overview
End-to-end data warehouse built on AWS cloud
for analyzing 541,909 retail transactions from
a UK-based e-commerce company.

## 🛠️ Tech Stack
- AWS S3 (Data Lake)
- Amazon Redshift Serverless (Data Warehouse)
- SQL (ETL + Analytics)
- Power BI (Visualization)

## 🏗️ Architecture
S3 → Redshift Staging → Star Schema → Power BI

## 📊 Key Results
- 541,909 transactions processed
- 25,900 unique customers
- UK highest revenue market
- 4 business analytics queries

## 📁 Project Structure
- 01_staging_table.sql — Raw data landing zone
- 02_star_schema.sql — Dimensional modeling
- 03_analytics_queries.sql — Business insights
